# Gresolutions

<div align="center">
  <p><strong>Soluciones basadas en datos y diseño digital para los desafíos modernos.</strong></p>
</div>

**Gresolutions** es un proyecto enfocado en brindar soluciones integrales, creado por **Santiago Greco** (Ingeniero Industrial y Analista de Datos) y **Martina Greco** (Diseñadora UX/UI). Combinamos el análisis de datos, el desarrollo tecnológico y el diseño centrado en el usuario para crear herramientas inteligentes, modulares y escalables que impulsan el crecimiento empresarial.

## 🚀 Servicios

- **📊 Soluciones con Datos**: Desde la limpieza y análisis de datos hasta la creación de dashboards e informes completos.
- **🤖 Herramientas con IA**: Desarrollo de modelos personalizados de inteligencia artificial para automatizar procesos y predecir resultados.
- **💻 Desarrollo de Web Apps**: Creación de aplicaciones web responsivas y fáciles de usar para dar vida a soluciones de datos e IA.
- **🎨 Landing Pages**: Diseño y desarrollo de páginas web atractivas y optimizadas para la conversión.

## 💼 Proyectos Destacados

- **[Cesto Tracker](https://cestotracker.com)**: WebApp para registrar partidos en vivo de Cestoball.
- **[Rota Pádel](https://rota-padel.netlify.app)**: Aplicación dinámica para gestionar y rotar jugadores en partidos de pádel de forma equitativa.
- **[Calculadora de Asado](https://asadito-site.netlify.app)**: Herramienta inteligente y rápida para calcular con precisión la cantidad de comida, bebida y carbón necesarios según la cantidad de invitados.
- **[RENO](https://i006-reno-arg-fullstack-1.onrender.com/login)**: Plataforma inteligente para la digitalización de procesos industriales, eliminando el uso de papel y centralizando datos técnicos en tiempo real.

## 🛠️ Tecnologías del Portfolio

Este sitio web fue construido utilizando herramientas modernas de desarrollo web:
- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS v4**

## 🏃‍♂️ Instalación y Uso Local

Para correr este proyecto localmente:

1. Clonar el repositorio.
2. Instalar las dependencias:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
